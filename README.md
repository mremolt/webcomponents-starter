# Webcomponents starter

Simple starter template for developing SPAs with Web Components. 
Just for demo purposes. 

## Installation

Checkout repo

``` 
npm install 
npm run serve
```

Visit http://localhost:4200/. 

## Production build

``` 
npm run build
npm run serve:prod
```

To show bundle info:

``` 
npm run stats
```

## Generate API Doc

``` 
npm run docs
start docs/index.html
```


## Run existing unit tests

``` 
npm run test
```
